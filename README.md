### 很早以前北京豌豆互动科技出品的一个仿《新剑侠情缘》的网页游戏，当时仅支持IE8及以下的浏览器，经过修改，目前支持IE10+、Safari、Chrome和FireFox
* 目前还存在一些bug，如敌人没血了还继续攻击、有时没法攻击敌人、进入下个地图判断区域太小导致没法切换地图等问题，欢迎提Pull Request。
